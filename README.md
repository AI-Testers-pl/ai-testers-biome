# AI Testers – Biome

Znajdziesz tutaj wszystkie informacje potrzebne do zainstalowania i korzystania z **Biome** – nowoczesnego narzędzia do formatowania i lintowania kodu JavaScript/TypeScript. Rozszerzenie tematów znajdziesz w kursie **(AI_Testers vol. 1 – Dodatki Zaawansowane)[https://jaktestowac.pl/course/ait1-bonusy-2/]** z programu **(AI_Testers)[https://aitesters.pl]**.

## Instalacja projektu

### Kroki instalacji

1. Sklonuj repozytorium:

```bash
git clone https://github.com/AI-Testers-pl/ai-testers-biome.git
cd ai-testers-biome
```

2. Zainstaluj zależności:

```bash
npm install
```

## Instalacja Biome

Jeśli chcesz dodać Biome do własnego projektu, użyj poniższej komendy:

```bash
npm install --save-dev @biomejs/biome
```

## Polecenia użyte w lekcji

### Formatowanie kodu

```bash
npx biome format --write
```

### Lintowanie + formatowanie

```bash
npx biome check --write --unsafe
```

## Porównanie z Prettier & ESLint

Jeśli chcesz porównać konfigurację Biome z klasycznym podejściem opartym na Prettier i ESLint, możesz przełączyć się na gałąź `prettier-eslint`:

```bash
git checkout prettier-eslint
```
